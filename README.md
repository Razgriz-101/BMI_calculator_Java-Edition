# BMI計算機
JavaFXとScene Builderを使用しBMI計算ができるGUIアプリを作成しました。\
入力欄へ身長と体重を半角数字で入力し「計算する」ボタンを押すとBMIと該当するカテゴリークラスが表示されます。
*****

| カテゴリー          | BMI    |
| ------------------ | ------ |
| 低体重（痩せすぎ）   | < 16.0 |
| 低体重（痩せ）       | < 17   |
| 低体重（痩せぎみ）   | < 18.5 |
| 普通体重            | < 25.0 |
| 肥満予備軍（過体重） | < 30.0 |
| 肥満（1度）         | < 35.0 |
| 肥満（2度）         | < 40.0 |
| 肥満（3度）         | ≧ 40.0 |
*****

**●注意**

BMIはあくまで身長と体重から導き出される目安です。
体脂肪率によって実際の肥満度は変わります。
